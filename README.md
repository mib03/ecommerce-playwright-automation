# Ecommerce Playwright Automation

Automation testing project using Playwright + TypeScript with Page Object Model.

## Tech Stack
- Playwright
- TypeScript
- Page Object Model
- Data Driven Testing

## Features Covered
- Login (positive & negative scenarios)
- Place Order end-to-end flow
- Validation scenarios

## Test Scenario Example

- User can login with valid credentials
- User cannot login with invalid password
- User can place order successfully

## Project Structure
tests/
  ui/
pages/
utils/
test-data/

## How to Run
npm install
npx playwright test

## Notes
This project demonstrates automation framework design and best practices.
